# 🏠 ViviendaMérida · Mérida XYZ

> **Buscador de viviendas e inmuebles en venta en el municipio de Mérida.**

**ViviendaMérida** es un portal inmobiliario de utilidad pública diseñado para simplificar la búsqueda de hogar en la ciudad. Mediante el uso de tecnologías de agregación de datos, el usuario puede consultar cientos de ofertas de múltiples portales locales en un único panel limpio, rápido y sin distracciones.

## 🚀 Características Principales

- **Monitor Inmobiliario Profundo**: Rastreo de hasta 10 páginas de resultados para capturar una imagen real del mercado local.
- **Filtro de Precio Inteligente**: Deslizador interactivo que permite filtrar propiedades en tiempo real (de 0€ a 1.000.000€).
- **Sistema de Desduplicación**: Algoritmo de "huella digital" (fingerprint) que detecta y elimina anuncios repetidos entre diferentes fuentes.
- **Interfaz Premium**: Diseño optimizado para móviles y escritorio, integrado en la estética minimalista de Mérida XYZ.

## 🛠️ Tecnologías Utilizadas

- **Scraper Engine**: Python con BeautifulSoup para la extracción y desduplicación de datos de portales inmobiliarios.
- **Frontend Layer**: HTML5, CSS3 avanzado y Javascript vanilla para una carga ultrarrápida.
- **Automation Pipeline**: GitHub Actions para la actualización automática diaria a las 06:00 AM.

## 📊 Fuentes de Datos

La información se agrega de los principales portales inmobiliarios nacionales (como Pisos.com) centrados en la ciudad de Mérida, procesando más de 250 inmuebles únicos para ofrecer una visión exhaustiva del mercado disponible.
